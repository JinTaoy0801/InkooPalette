Inkoo's Palette

Game Design Document

By Devon Lum, Jin Tao Yu, and Kevin Wang

Introduction

This document describes a game called Inkoo's Palette, designed to be a simple, 2D platformer, constructed and documented to help others learn to make simple, 2D platformers. The game will employ sprite-based animation, tiled backgrounds, collision detection, physics, AI, platformer & gravity, efficient memory management, and more basic 2D game techniques.

Technology

Inkoo's Palette will be developed for the Windows Platform using the Wolfie2D game engine, which is a game engine developed by Joe Weaver and Professor Richard McKenna in Fall 2020 for Stony Brooks 2D game programming course as an alternative to Phaser. The game will be coded in Typescript. Audacity will be used for all music and sound effects management. Artwork will be drawn using Piskel.

Backstory

Once there was a painter who was beloved by colors, he who gave the world color had three disciples, whom he gave each a primary color. The painter wanted his disciples to venture throughout the world to spread out their color evenly, so all corners of the world can be colorful. But they got selfish and wanted to keep the color all to themselves. In a world absence of color, a single spirit of color emerges to restore color back into the world. (subjective to change)

Objective

The spirit of color, Inkoo, emerges to restore color to the world; he must defeat the three disciples and restore color back to the world.

Gameplay

This game will work like a platformer where Inkoo will be able to move left and right, and jump on platforms and other surfaces. Inkoo will have attack abilities with set ranges to hit enemies.

Inkoo would have a three part attack, where the player is rewarded if he makes it up to the third string and is able to hit it before the enemies hit him back since it would deal more damage. The player is not forced to do so though.

  • Inkoo would have an unlockable dashing ability after he complete said [objective]
  • Inkoo would have an unlockable range attack ability after he complete said [objective]
  • Inkoo would have an unlockable double jump ability after he complete said [objective]

Controls

This game requires a keyboard only and has the following controls:

  • W/Spacebar - Jump
  • A - Move Left
  • S - Quick Fall
  • D - Move Right
  • Left Shift - Dash
  • J - Basic Attack
  • K - Range attack
  • Esc - Pause Game

Graphical User Interface

As far as the GUI is concerned, if we consider

  • Splash Screen - The splash screen GUI simply presents the game name and the game logo. Pressing any button will bring the player to the Game Menu.



  • Game Menu - After the player presses any button from the Splash Screen, it leads the player to this menu where they will be able to select from the following options:



    • Start - Upon selecting Start, the game should start immediately
    • Controls - Upon selecting Controls, it would display all game controls



    • Story - Upon selecting Story, it would display a page explaining a bit of the game's story



    • Exit - Upon selecting Exit, the game would close

  • In-Game GUI - While playing the following will be displayed at all times:

    • Ink Hearts - Amount of hearts remaining


  • Pause Menu - The menu that appears when the player presses esc during the game. The pause menu will have the following options:

    • Continue - Selecting this button will allow the player to continue their game
    • Controls - When pressed, the game will display the Controls Menu
    • Exit - When pressed, the game will exit and return to the Splash Screen

Artwork

All artwork in the game will be original. The following needs to be created:

  • Inkoo - Inkoo's sprite will need the following animations:
    • Moving Left
    • Moving Right
    • Jumping
    • Jumping Left
    • Jumping Right
    • Falling
    • Falling Left
    • Falling Right
    • Attacking
    • Dying
    • Dead

  • Enemies
    • Moving Left
    • Moving Right
    • Attacking Left
    • Attacking Right

  • Ceiling, Floor, and Wall Tiles

  • World Background

Sound Effects

All sound effects will be original. Sounds must be made to coincide with each of the following events:

  • Jump
  • Inkoo Dying
  • Attacking
  • Monster Attacking

Music
